Kebijakan Privasi

Berlaku sejak: 8 Januari 2026 | Versi: 1.0.4

Selamat datang di IPTV ID ("Aplikasi"). Privasi pengguna penting bagi kami. Dokumen ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda ketika Anda menggunakan Aplikasi ini.

Kepatuhan GDPR: Aplikasi ini sepenuhnya mematuhi General Data Protection Regulation (GDPR) Uni Eropa. Jika Anda berada di Uni Eropa atau EEA, kami akan meminta persetujuan Anda sebelum mengumpulkan data untuk iklan yang dipersonalisasi.

1. Informasi yang Kami Kumpulkan

Kami berupaya meminimalkan data yang dikumpulkan. Jenis data yang mungkin dikumpulkan meliputi:

  • Data yang Anda berikan secara langsung: seperti playlist atau URL M3U yang Anda tambahkan ke Aplikasi, nama playlist yang Anda simpan, dan data favorit (channel yang Anda tandai sebagai favorit). Semua item ini disimpan secara lokal di perangkat Anda kecuali jika Anda memilih fitur sinkronisasi (jika tersedia di masa depan).
  • Data teknis dan penggunaan: informasi tidak pribadi seperti jenis perangkat, versi OS, statistik penggunaan aplikasi (mis. fitur yang dipakai), dan informasi crash yang membantu kami memperbaiki bug. Informasi ini dikumpulkan menggunakan layanan analitik dan crash reporting yang disebutkan di bawah.
  • Data periklanan: informasi yang memungkinkan penyajian iklan yang relevan (mis. ID iklan perangkat/Advertising ID). Aplikasi menampilkan iklan melalui jaringan iklan pihak ketiga. Untuk pengguna di Uni Eropa/EEA, kami menggunakan Google User Messaging Platform (UMP) untuk meminta persetujuan Anda sebelum mengumpulkan data iklan.
  • Publisher-Provided ID: Kami menghasilkan ID unik (UUID) yang tidak mengandung informasi pribadi untuk memenuhi persyaratan Google AdMob GDPR.

2. Tujuan Penggunaan Data

Kami menggunakan data untuk tujuan berikut:

  • Menjalankan dan menyediakan fungsi Aplikasi (memutar channel, menyimpan playlist/favorit secara lokal, dsb.).
  • Meningkatkan kualitas layanan, mendiagnosis dan memperbaiki masalah teknis (melalui crash reporting dan analitik).
  • Menyajikan iklan yang relevan apabila Aplikasi menampilkan iklan.
  • Memenuhi kewajiban hukum atau permintaan sah dari instansi berwenang apabila diperlukan.

3. Penyimpanan Data

Data playlist, preferensi, dan favorit disimpan secara lokal di perangkat Anda menggunakan storage lokal (tidak kami unggah ke server kami secara otomatis). Data teknis dan crash dapat dikirim ke layanan pihak ketiga (mis. Firebase Crashlytics, Firebase Analytics) sesuai dengan bagian Pihak Ketiga di bawah.

4. Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga berikut untuk fungsi tertentu:

  • Firebase (Google) — untuk analitik dan pelaporan crash (Firebase Crashlytics, Firebase Analytics, Firebase Messaging). Layanan ini dapat mengumpulkan data perangkat dan penggunaan yang bersifat teknis. Silakan lihat kebijakan privasi Google: https://policies.google.com/privacy.
  • Google Mobile Ads / AdMob — menampilkan iklan di dalam Aplikasi. Kami menggunakan Google User Messaging Platform (UMP) untuk mematuhi GDPR dan meminta persetujuan pengguna di Uni Eropa sebelum menampilkan iklan yang dipersonalisasi. Kebijakan privasi AdMob: https://support.google.com/admob/answer/6128543.
  • AppLovin — jaringan iklan mediasi untuk menampilkan iklan. Kebijakan privasi: https://www.applovin.com/privacy/.
  • Unity Ads — jaringan iklan mediasi untuk menampilkan iklan. Kebijakan privasi: https://unity.com/legal/privacy-policy.
  • IronSource — jaringan iklan mediasi untuk menampilkan iklan. Kebijakan privasi: https://developers.is.com/ironsource-mobile/air/ironsource-mobile-privacy-policy/.
  • Mintegral — jaringan iklan mediasi untuk menampilkan iklan. Kebijakan privasi: https://www.mintegral.com/en/privacy/.
  • Media Kit — pustaka pemutar video untuk streaming konten IPTV (HLS/M3U8/DASH). Pustaka ini hanya untuk memutar konten dan tidak mengirim data playlist Anda ke server.
Catatan penting tentang GDPR:
  • Kami tidak menjual data pribadi Anda kepada pihak ketiga.
  • Semua data playlist dan favorit disimpan secara lokal di perangkat Anda.
  • Untuk pengguna di Uni Eropa/EEA, kami meminta persetujuan Anda melalui Google UMP sebelum menampilkan iklan yang dipersonalisasi.
  • Anda dapat mengubah preferensi persetujuan iklan kapan saja melalui Pengaturan → Privasi di aplikasi.
  • Publisher-Provided ID yang kami gunakan adalah UUID acak dan tidak mengandung informasi pribadi apapun.

5. Izin yang Diperlukan

Tergantung pada fitur yang Anda gunakan, Aplikasi mungkin meminta izin di perangkat Anda, contohnya:

  • Akses Internet — diperlukan untuk streaming konten dan memuat iklan.
  • Akses Storage / File — jika Anda memilih mengunggah file M3U dari perangkat, Aplikasi akan meminta akses baca file (bergantung pada platform dan versi OS).

6. Keamanan

Kami menerapkan langkah-langkah teknis wajar untuk membantu melindungi data Anda. Harap diperhatikan bahwa tidak ada metode transmisi atau penyimpanan elektronik yang 100% aman. Jika Anda menyimpan playlist berisi kredensial atau tautan privat, pertimbangkan risikonya sebelum menambahkannya ke Aplikasi.

7. Hak Pengguna

Anda memiliki hak penuh atas data Anda:

  • Akses dan Pengelolaan Data Lokal: Anda dapat melihat, mengedit, dan menghapus playlist, favorit, dan pengaturan kapan saja melalui aplikasi.
  • Hak GDPR (untuk pengguna Uni Eropa/EEA):
    • Hak untuk mengetahui: Anda berhak mengetahui data apa yang dikumpulkan tentang Anda.
    • Hak untuk mengakses: Anda dapat meminta salinan data pribadi yang kami simpan.
    • Hak untuk mengoreksi: Anda dapat meminta koreksi data yang tidak akurat.
    • Hak untuk menghapus: Anda dapat meminta penghapusan data pribadi Anda ("hak untuk dilupakan").
    • Hak untuk membatasi pemrosesan: Anda dapat meminta pembatasan pemrosesan data Anda.
    • Hak untuk portabilitas data: Anda dapat meminta data Anda dalam format yang dapat dibaca mesin.
    • Hak untuk menolak: Anda dapat menolak pemrosesan data tertentu, termasuk iklan yang dipersonalisasi.
    • Hak untuk menarik persetujuan: Anda dapat menarik persetujuan Anda kapan saja melalui Pengaturan → Privasi di aplikasi.
  • Pengelolaan Persetujuan Iklan: Anda dapat mengubah preferensi persetujuan iklan melalui menu Pengaturan aplikasi. Untuk pengguna di Uni Eropa, Anda dapat mengubah pilihan Anda tentang iklan yang dipersonalisasi kapan saja.
  • Opt-Out dari Iklan yang Dipersonalisasi:
    • Android: Buka Pengaturan → Google → Iklan → Nonaktifkan personalisasi iklan
    • iOS: Buka Pengaturan → Privasi → Iklan Apple → Batasi Pelacakan Iklan

Untuk menggunakan hak GDPR Anda atau jika memiliki pertanyaan, silakan hubungi kami di alamat email yang tercantum di bagian Kontak.

8. Anak-anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan data pribadi dari anak-anak. Jika Anda percaya bahwa kami telah mengumpulkan informasi dari anak di bawah usia yang berlaku, silakan hubungi kami untuk meminta penghapusan.

9. Perubahan Kebijakan

Kami dapat mengubah kebijakan ini dari waktu ke waktu. Jika terjadi perubahan material, kami akan menampilkan pemberitahuan di aplikasi atau memperbarui tanggal efektif di bagian atas dokumen ini.
